# Hebi audio

Files are bundled in the repo. **Hard refresh** (`Cmd+Shift+R`) after updates, then turn sound on in the dock (Hebi / light theme).

| File | Role |
|------|------|
| `ambient.mp3` | Background loop |
| `click-soft.wav` | General UI clicks |
| `click-nav.wav` | Dock navigation |
| `click-toggle.wav` | Sound on/off button |

**Optional:** Replace `ambient.mp3` with your Pixabay download ([Ambient Electronics](https://pixabay.com/music/ambient-ambient-electronics-524300/)) and bump `HEBI_AUDIO_VERSION` in `data/site.js`.
